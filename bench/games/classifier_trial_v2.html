How it works

Each round shows one user prompt and three anonymized model responses labeled A, B, and C.

  • One response used the three-clause chain.
  • One used absurd nonsense instructions.
  • One used the default helpful-assistant system prompt.
  • You pick the response that handles the situation best.
  • On mobile, each response gets a focused reading view so long answers do not disappear into tiny scroll boxes.

What this is testing

Payload, not prose The point is whether the chain's specific content beats both nonsense and baseline in blind comparison.
Fast enough to spread The page is static, preloaded, and designed to start instantly.
Shareable outcome At the end you get a clean score card you can screenshot or share directly.

All 17 rounds are bundled into this page. No runtime API call. No login. No waiting.
